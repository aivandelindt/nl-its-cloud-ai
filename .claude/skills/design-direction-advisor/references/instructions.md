1. "Create a presentation on AI Psychology, recommend 3 style directions for me to choose from."

The user wants a slide deck on a cross-disciplinary topic. The smart part is the "recommend 3 style directions" — instead of guessing blindly, the AI first offers curated options (e.g., academic minimal, futuristic dark, illustrative informal). The user picks one, then the AI generates the full deck. This creates a choose-then-execute workflow that saves back-and-forth.

1. "Build an AI Pomodoro Timer iOS prototype, 4 core screens with real clickable interactions."

A full interactive prototype — not just static mockups. The "4 core screens" likely cover: timer running, task list, focus stats, settings. The key requirement is real clickability — screen transitions, button taps, state changes all work. This is essentially a functional prototype you could hand to a designer or developer as a spec.

1. "Turn this logic into a 60-second animation, export as MP4 and GIF."

The user has some process, algorithm, or flow — and wants it visualized as a timed animation. The 60-second constraint means clear pacing, probably with an intro → body → conclusion structure. Dual export (MP4 for quality, GIF for embedding in docs/slides) shows they're thinking about different use contexts. The hard part here is that producing true video assets usually requires a toolchain beyond chat generation.

1. "Help me review this design across 5 dimensions."

A structured critique framework. The user wants more than "looks good" — they want systematic evaluation. Common 5 dimensions include: Usability, Visual Design, Accessibility, Consistency, and Feasibility (or alternatively: Clarity, Aesthetics, Efficiency, Error Tolerance, Delight). This turns AI into a design review partner that gives actionable, categorized feedback.

The Common Thread
These aren't simple Q&A prompts — they're production briefs that:

Ask for a structured artifact (slides, prototype, animation, critique)
Set concrete constraints (3 options, 4 screens, 60 seconds, 5 dimensions)
Build in a feedback loop (choose first, then execute)
Consider the output medium (MP4 vs GIF, clickable vs static)
Each one treats the AI not as a chatbot, but as a creative collaborator with specific capabilities — a designer, a prototyper, an animator, or a design reviewer.
